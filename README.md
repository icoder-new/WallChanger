# WallChanger (only for Windows)

## About
Simple and easy2use app to change your wallpaper on CMD. GUI version comming soon!

## How to install and use

### Install
```bash
git clone https://github.com/icoder-new/WallChanger
cd ./WallChanger
```

### Using
```bash
py main.py
```
